Operações com Vetores - Adição e Subtração

O que são Ventores

  • Vetores são entidades matemáticas que representam magnitude e direção. Eles são amplamente utilizados em física, matemática, ciência da computação e várias outras áreas.

  • Em termos simples, um vetor é uma quantidade que possui tanto magnitude (ou tamanho) quanto direção. Por exemplo, ao descrever o movimento de um objeto, você não só precisa saber a rapidez com que ele se move (magnitude), mas também em que direção está se movendo.

  • Os vetores podem ser representados graficamente por setas, onde o comprimento da seta representa a magnitude e a direção da seta indica a direção do vetor. Por exemplo, em um gráfico bidimensional, um vetor pode ser representado como uma seta que se estende de um ponto de origem para um ponto final.

  • Além disso, os vetores podem ser somados, subtraídos, multiplicados por escalares (números reais) e têm outras operações definidas, o que os torna uma ferramenta poderosa em diversas áreas da matemática e das ciências aplicadas.e eficiente.

Em ciência de dados, os vetores são usados em várias etapas do processo de análise e modelagem de dados. Aqui estão alguns dos principais usos de vetores em ciência de dados:

  • Representação de dados: Os dados muitas vezes são representados como vetores, onde cada componente do vetor corresponde a uma característica ou atributo do dado. Por exemplo, em uma base de dados de clientes, cada cliente pode ser representado por um vetor onde cada componente representa uma característica como idade, renda, histórico de compras, etc.

  • Algoritmos de Machine Learning: Muitos algoritmos de machine learning, como regressão linear, máquinas de vetores de suporte (SVM), redes neurais, entre outros, trabalham com vetores como entrada. Esses algoritmos podem aprender padrões e fazer previsões com base nos vetores de entrada.

  • Processamento de linguagem natural (NLP): Em NLP, palavras, frases ou documentos são muitas vezes representados como vetores em espaços vetoriais, como o espaço vetorial de palavras (word embeddings). Esses vetores capturam as relações semânticas e sintáticas entre as palavras e são usados em tarefas como classificação de texto, tradução automática e análise de sentimento.

  • Redução de dimensionalidade: Em conjuntos de dados com muitas características, técnicas de redução de dimensionalidade, como Análise de Componentes Principais (PCA) e T-Distributed Stochastic Neighbor Embedding (t-SNE), são frequentemente usadas para representar os dados em um espaço de menor dimensionalidade sem perder muita informação. Isso é útil para visualização e simplificação de modelos.

  • Visualização de dados: Técnicas de visualização de dados, como gráficos de dispersão (scatter plots) ou mapas de calor (heatmaps), muitas vezes representam os dados como vetores em espaços bidimensionais ou tridimensionais para que possam ser visualizados de forma eficaz.

Em resumo, os vetores são uma ferramenta fundamental em ciência de dados, permitindo a representação, análise, modelagem e visualização de dados de forma eficaz e eficiente.

# Definindo dois vetores como listas
vetor_a = [3, 4, 5]
vetor_b = [1, 2, 3]
# Função para adição de vetores
def adicionar_vetores(a, b):
    return [a[i] + b[i] for i in range(len(a))]
# Função para subtração de vetores
def subtrair_vetores(a, b):
    return [a[i] - b[i] for i in range(len(a))]
# Realizando as operações
vetor_soma = adicionar_vetores(vetor_a, vetor_b)
vetor_diferenca = subtrair_vetores(vetor_a, vetor_b)
# Mostrando os resultados
print("Vetor A:", vetor_a)
print("Vetor B:", vetor_b)
print("Soma dos Vetores:", vetor_soma)
print("Diferença dos Vetores:", vetor_diferenca)

Operações com Vetores - Cross Product (Produto Vetorial)

  • O produto vetorial entre dois vetores resulta em um terceiro vetor que é perpendicular aos dois vetores originais.
# Função para calcular o produto vetorial entre 2 vetores
def produto_vetorial(v1, v2):
    x = v1[1] * v2[2] - v1[2] * v2[1]
    y = v1[2] * v2[0] - v1[0] * v2[2]
    z = v1[0] * v2[1] - v1[1] * v2[0]
    return [x, y, z]
# Cria os vetores
vetor_a = [2, -3, 1]
vetor_b = [5, 7, 6]
# Calculando o produto vetorial
resultado = produto_vetorial(vetor_a, vetor_b)
print("Produto Vetorial:", resultado)
Produto Vetorial: [-25, -7, 29]

Operações com Vetores - Dot Product (Produto Escalar)

  • O produto escalar de dois vetores é uma operação que resulta em um escalar (um número). Matematicamente, é a soma dos produtos dos componentes correspondentes dos vetores.
# Função para calcular o produto escalar
def produto_escalar(v1, v2):
    return sum(a * b for a, b in zip(v1, v2))
# Cria os vetores
vetor_a = [2, -3, 1]
vetor_b = [5, 7, 6]
# Calculando o produto escalar
resultado = produto_escalar(vetor_a, vetor_b)
print("Produto Escalar:", resultado)

Função de Custo MSE com Operações Entre Vetores

  • Esta função dsa_func_custo é uma implementação da função de custo de erro quadrático médio, que é amplamente usada em regressão linear.
  • O objetivo dessa função é medir quão bem o modelo de regressão (representado pelos parâmetros theta) está desempenhando em relação ao conjunto de dados (X e y).
    • Um custo mais baixo indica que o modelo está fazendo previsões mais precisas.
# Função de Custo MSE com Operações Entre Vetores
def fn_func_custo(X, y, theta):
    
    # m é o número de exemplos de treinamento (observações) no conjunto de dados
    m = len(y)

    # Calcula a função de custo para a regressão linear
    # A função de custo é a média dos erros quadrados entre as previsões e os valores reais
    # (1/(2*m)) é um fator de escala comum na função de custo do erro quadrático médio
    # np.sum(np.square(X.dot(theta) - y)) é a soma dos quadrados dos erros
    # X.dot(theta) é o produto escalar de X (dados de entrada) e theta (parâmetros do modelo)
    # X.dot(theta) - y é a diferença entre as previsões e os valores reais
    cost = (1/(2*m)) * np.sum(np.square(X.dot(theta) - y))

    # Retorna o custo calculado
    return cost

Algoritmo Gradiente Descendente com Operações entre Vetores 

# Gradiente descendente
def fn_gradient_descent(X, y, theta, learning_rate, iterations):
    
    # m é o número de exemplos de treinamento no conjunto de dados
    m = len(y)

    # Inicializa um array para armazenar o histórico de custo em cada iteração
    cost_history = np.zeros(iterations)

    # Loop para realizar o número especificado de iterações
    for i in range(iterations):
        
        # Calcula o gradiente da função de custo em relação a theta
        # O gradiente é a média dos produtos dos erros e as características de entrada
        gradients = 1/m * X.T.dot(X.dot(theta) - y)

        # Atualiza os parâmetros theta subtraindo o produto da taxa de aprendizado e o gradiente
        # Esta é a etapa de descida do gradiente, movendo-se em direção ao mínimo da função de custo
        theta = theta - learning_rate * gradients

        # Calcula o custo com o theta atualizado e armazena no histórico de custo
        cost_history[i] = fn_func_custo(X, y, theta)
    
    # Retorna os parâmetros theta otimizados e o histórico de custo
    return theta, cost_history
