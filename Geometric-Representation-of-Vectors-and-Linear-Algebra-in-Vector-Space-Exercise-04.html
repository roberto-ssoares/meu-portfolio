Exercício 1: Produto Escalar em Aplicação Prática

  • Descrição: Dados dois vetores que representam preferências de usuários em três categorias diferentes user1 = [4, 3, 2] e user2 = [1, 5, 4], calcule
    a similaridade entre esses usuários usando o produto escalar. 
# Vetores representando as preferências dos usuários
user1 = [4, 3, 2]
user2 = [1, 5, 4]

# Função para calcular o produto escalar entre dois vetores
def produto_escalar(vetor1, vetor2):
    # Verifica se os vetores têm o mesmo tamanho
    if len(vetor1) != len(vetor2):
        raise ValueError("Os vetores devem ter o mesmo tamanho.")
    
    # Inicializa o produto escalar como 0
    escalar = 0
    
    # Calcula o produto escalar somando o produto de cada par de elementos
    for i in range(len(vetor1)):
        escalar += vetor1[i] * vetor2[i]
    
    return escalar

# Calcula o produto escalar entre os vetores user1 e user2
resultado_produto_escalar = produto_escalar(user1, user2)

# Exibe o resultado
print("O produto escalar entre os usuários é:", resultado_produto_escalar)
O produto escalar entre os usuários é: 27

Exercício 2: Multiplicação de Escalar por Vetor em Dados Reais

  • Descrição: Dada uma série de preços de ações prices= [120, 125, 130, 128, 135], ajuste os preços em 15% e calcule o novo preço das ações. 
# Série de preços de ações
prices = [120, 125, 130, 128, 135]

# Porcentagem de ajuste
percentagem = 0.15

# Função para ajustar os preços em uma certa porcentagem
def ajustar_precos(prices, percentagem):
    # Lista para armazenar os novos preços ajustados
    novos_precos = []
    
    # Itera sobre os preços originais e ajusta cada um
    for price in prices:
        novo_preco = price * (1 + percentagem)  # Calcula o novo preço
        novos_precos.append(novo_preco)  # Adiciona o novo preço à lista
        
    return novos_precos

# Chama a função para ajustar os preços
novos_precos = ajustar_precos(prices, percentagem)

# Exibe os novos preços ajustados
print("Novos preços ajustados em 15%:")
for i, novo_preco in enumerate(novos_precos):
    print(f"Ação {i+1}: ${novo_preco:.2f}")

Exercício 3: Otimização de Produto Escalar

  • Descrição: Encontre o vetor unitário que maximiza o produto escalar com o vetor v = [3, 4, 5]. Use técnicas de otimização para encontrar a solução. 
from scipy.optimize import minimize

# Função objetivo
def objective_function(u):
    v = np.array([3, 4, 5])
    return -np.dot(u, v)  # Minimizar o negativo do produto escalar para maximizar o produto escalar

# Restrição de comprimento unitário
def constraint(u):
    return np.sum(u**2) - 1

# Vetor de condições iniciais
initial_guess = np.array([1, 0, 0])  # Começando com [1, 0, 0] é uma escolha razoável

# Definindo a restrição
constraint_definition = {'type': 'eq', 'fun': constraint}

# Otimização usando o método dos multiplicadores de Lagrange
result = minimize(objective_function, initial_guess, constraints=constraint_definition)

# Vetor unitário que maximiza o produto escalar
unit_vector_max_scalar_product = result.x

# Produto escalar máximo
max_scalar_product = -result.fun

print("Vetor unitário que maximiza o produto escalar:", unit_vector_max_scalar_product)
print("Produto escalar máximo:", max_scalar_product)

Exercício 4: Produto Vetorial e Direção Normal

  • Descrição: Dados três pontos no espaço 3D
    • A = [1, 2, 3], B = [4, 5, 6], e C = [7, 8, 9],
    • encontre um vetor normal ao plano definido por esses três pontos. 
import numpy as np

# Definindo os pontos A, B e C
A = np.array([1, 2, 3])
B = np.array([4, 5, 6])
C = np.array([7, 8, 9])

# Vetores que estão no plano
AB = B - A
AC = C - A

# Calculando o produto vetorial
vetor_normal = np.cross(AB, AC)

print("Um vetor normal ao plano definido por A, B e C é:", vetor_normal)
Um vetor normal ao plano definido por A, B e C é: [0 0 0]

Exercício 5: Produto Vetorial em Análise de Dados

  • Descrição: Em um conjunto de dados 3D com pontos p1 = [1, 2, 3], p2 = [4, 5, 6], e p3 = [7, 8, 9], encontre o vetor normal ao
    plano formado por esses três pontos e interprete seu significado em termos de orientação do plano.

import numpy as np

# Definindo os pontos p1, p2 e p3
p1 = np.array([1, 2, 3])
p2 = np.array([4, 5, 6])
p3 = np.array([7, 8, 9])

# Vetores que estão no plano
p1p2 = p2 - p1
p1p3 = p3 - p1

# Calculando o produto vetorial
vetor_normal = np.cross(p1p2, p1p3)

# Interpretando a orientação do plano
if np.allclose(vetor_normal, [0, 0, 0]):
    print("Os pontos estão alinhados, não formando um plano.")
elif np.all(vetor_normal > 0):
    print("O plano tem orientação para fora do espaço delimitado pelos pontos.")
elif np.all(vetor_normal < 0):
    print("O plano tem orientação para dentro do espaço delimitado pelos pontos.")
else:
    print("O plano não tem uma orientação clara em relação ao espaço delimitado pelos pontos.")

print("O vetor normal ao plano formado pelos pontos é:", vetor_normal)
Os pontos estão alinhados, não formando um plano. O vetor normal ao plano formado pelos pontos é: [0 0 0]
