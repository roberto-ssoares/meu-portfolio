Exercício 1: Cálculo de Área de um Círculo

  • Escreva um programa que calcula a área de um círculo.
  • O usuário deve fornecer o raio
  • e o programa deve retornar a área.

#A = π r²

# Importar o módulo math para acessar constantes e funções matemáticas, como o valor de pi
import math

def area_circulo(raio):
    # Calcula a área do círculo usando a fórmula área = π * raio^2
    # math.pi fornece o valor de π e raio ** 2 calcula o quadrado do raio
    return math.pi * raio ** 2

# Pedir ao usuário para inserir o valor do raio do círculo e converter essa entrada em um número de ponto flutuante "mais precisão"
raio = float(input("Digite o raio do círculo: "))

# Imprimir a área do círculo chamando a função area_circulo com o raio fornecido e exibindo o resultado
print("Área do círculo: ",area_circulo(raio))

# Multiplicar a temperatura em Celsius por 9/5 e adiciona 32 para converter para Fahrenheit
def celsius_para_fahrenheit(c):
    return (c * 9/5) + 32
    
# Subtrair 32 da temperatura em Fahrenheit e multiplica por 5/9 para converter para Celsius
def fahrenheit_para_celsius(f):
    return (f - 32) * 5/9

# Solicitar ao usuário para inserir uma temperatura e converter a entrada para um número de ponto flutuante
temp = float(input("Digite a temperatura: "))

    # Se for Celsius, converter a temperatura para Fahrenheit e imprime o resultado
    print("Temperatura em Fahrenheit: ", celsius_para_fahrenheit(temp))
else:
    # Se não for Celsius (assumindo Fahrenheit), converte a temperatura para Celsius e imprime o resultado
    print("Temperatura em Celsius:", fahrenheit_para_celsius(temp))

# 3. Criar função para resolver a equação linar ax + b = 0
def revolve_linear(a, b):
    # 3.1. Verificar se 'a' é zero, o que indica uma equação linear degenerada
    if a == 0:
        
        # 3.2. Se 'b' também for zero, a equação é determnada com soluções infinitas
        # caso contrário, é uma contradição e não tem solução
        return "Sem solução" if b != 0 else "Soluções infinitas"
        
    # 3.3. Se 'a' não é zero, revolve a equação linear ax + b = 0 e retorna a solução x = -b/a
    return -b/a

Exercício 5: Determinar as Raízes de uma Equação Quadrática

  • Crie um programa que encontra as raízes de uma equação quadrática do tipo ax² + bx + c = 0.
  • O programa deve lidar com todos os tipos de raízes (reais e complexas).

# 4. Criar uma função que realize o cálculo da equação quadrática
# 4.1. Importar o módulo cmath para operações matemáticas com números complexos
import cmath

def raizes_quadraticas(a, b, c):

    # 4.2. Calcula o discriminante (delta) da equação quadrática usando a fórmula delta = b^2 - 4ac
    #      cmath.sqrt é usado para lidar com raízes quadraticas de números negativos (números complexos)
    delta = cmath.sqrt(b**2 - 4*a*c)

    # 4.3. Calcula a primeira raiz (x1) da equação quadrática usando a fórmula x = (-b + sqtr(delta)) / 2a
    x1 = (-b + delta) / (2*a)

    # 4.4. Calcula a primeira raiz (x2) da equação quadrática usando a fórmula x = (-b + sqtr(delta)) / 2a
    x2 = (-b + delta) / (2*a)

    # 4.5. Retorna ambas as raízes como um par de valores
    return (x1, x2)

# 1. Solicitar ao usuário para inserir o valor do coeficiente 'a' e converter para um número de ponto flutuante
a = float(input("Digite o valor de a: "))

# 2. Solicitar ao usuário para inserir o valor do coeficiente 'b' e converter para um número de ponto flutuante
b = float(input("Digite o valor de b: "))

# 3. Solicitar ao usuário para inserir o valor do coeficiente 'c' e converter para um número de ponto flutuante
c = float(input("Digite o valor de c: "))

# Final. Imprime as raízes da equação quadrática, chamando a função raizes_quadraticas com os valores de 'a', 'b', e 'c'
print("As raízes da equação são: ", raizes_quadraticas(a, b, c))

Exercício 6: Calcular o Fatorial de um Número

Escreva uma função que calcula o fatorial de um número inteiro fornecido pelo usuário.

  • O fatorial de um número inteiro não negativo n, denotado por n!, é o produto de todos os inteiros positivos menores ou a n.
  • O conceito de fatorial é amplamente utilizado em matemática, especialmente em combinações e permutações, onde é usado para calcular o número de maneiras possíveis de organizar ou escolher objetos. 
# 2. Criar uma função para o cálculo do fatorial
def fatorial(n):

    # 2.1. Retorna 1 se n é 0, pois o fatorial de 0 é definido como 1
    #      Caso contrário, retorna n multiplicado pelo fatorial de n-1 (chamada recursiva),
    #      o que efetivamente constrói o produto n * (n-1) * (n-2) * ... * 1 
    return 1 if n == 0 else n * fatorial(n - 1)
    
# 1. Solicitar ao usuário para inserir o valor e converter para um número inteiro
num = int(input("Digite um número inteiro: "))

# Final. Imprimir o fatorial do número inserido, chamando a função fatorial
print("Fatorial: ", fatorial(num))

Exercício 7: Sequência de Fibonacci

  • Crie um programa que gera a sequência de Fibonacci até um determinado número n fornecido pelo usuário. 
# 2. Criar uma função para encontrar a sequência de Fibonacci
def fibonacci(n):

    # 2.1. Inicializa as duas primeiras variáveis, a e b, que armazenam os dois primeiros números da sequência de Fibonacci
    a, b = 0, 1

    # 2.2. Um loop que itera n vezes para gerar n termos da sequência
    for _ in range(n):

        # 2.2.1. Gera o valor atual de a, que é um termo da sequência de Fibonacci
        yield a

        # 2.2.2. Atualiza os valores de a e b para os próximos dois termos na sequência
        #        a recebe o valor atual de b, e b recebe a soma dos valores anteriores de a e b
        a, b = b, a + b

# 1. Solicitar ao usuário para inserir o número de termos desejados na sequência de Fibonacci e converter a entrada para um inteiro
n = int(input("Digite o número de termos da sequência de Fibonacci: "))

# Final. Imprime a sequência de Fibonacci como uma lista, convertendo o gerador de uma lista para exibir todos os termos gerados.
print("Sequência de Fibonacci: ", list(fibonacci(n)))

Exercício 8: Cálculo de Juros Compostos

  • Faça um programa que calcula o montante final de um investimento com juros compostos.
  • O usuário deve fornecer o capital inicial, a taxa de juros anual, e o número de anos.

# 4. Criar uma função para o cálculo do juros compostos
def juros_compostos(capital, taxa, anos):

    # 4.1. Calcula o montante final usuando a fórmula de juros compostos: M = C * (1 + r)^t
    #      onde M é o montante, C é o capital inicial, r é a taxa de juros, e t é o tempo em anos
    return capital * (1 + taxa) ** anos

# 1. Solicitar ao usuário para inserir o capital inicial e converter a entrada em um número de ponto flutuante
capital = float(input("Digite o capital inicial: "))

# 2. Solicitar ao usuário para inserir a taxa de juros anual (em formato decimal) e converter a entrada em um número de ponto flutuante
#    Por exemplo, uma taxa 5% deve ser inserida como 0.05
taxa = float(input("Digite a taxa de juros anual (em decimal): "))

# 3. Solicitar ao usuário para inserir o número de anos e converter a entrada em um número inteiro
anos = int(input("Digite o número de anos: "))

# Final. Imprime o montante final após o período especificado, chamando a função juros_compostos
#        com os valores de capital, taxa e juros.
print("Montante final: ", juros_compostos(capital, taxa, anos))

Exercício 10: Verificar se um Número é Primo

  • Crie uma função que verifica se um número fornecido pelo usuário é primo ou não.
    • Um número primo é um número natual maior do que 1 que não tem outros divisores além de 1 e ele mesmo.
    • Em outras palavras, um número é primo se ele só pode ser dividido de forma exata por 1 e por ele próprio.
    • Esse números têm uma importância fundamental em vária áreas da matemática, incluindo a teoria dos números, a criptografia e a análise de algoritmos 
# 2. Criar uma função para verificar se um número é primo
def e_primo(num):

    # 2.1. Percorre os números de 2 até a raiz quadrada do número (arredondamento para cima).
    #      Essa otimização é possível porque se um número tem um divisor maior que sua raiz quadrada, 
    #      então ele também deve ter um divisor menor que a raiz quadrada.
    for i in range(2, int(num**0.5) + 1 ):

        # 2.1.1. Se o número num for divisível por i, então ele não é primo
        if num % i == 0:
            return False

    #      Se o loop for concluído sem encotrar divisores, o número é primo
    return True

# 1. Pede ao usuário para digitar um número e converte a entrada em um inteiro
numero = int(input("Digite um número: "))

# Final. Imprime se o número é primo ou não, utilizando a função e_primo
print("O número é primo! " if e_primo(numero) else "O número não é primo! ")
